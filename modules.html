IDecider

IDecider<C, S, E>: I_Decider<C, S, S, E, E>
  • Defined in lib/domain/decider.ts:52

IDecider represents the main decision-making algorithm. It has five generic parameters C, S, E , representing the type of the values that IDecider may contain or use. IDecider can be specialized for any type C or S or E because these types does not affect its behavior. IDecider behaves the same for C=Int or C=YourCustomType, for example.

IDecider is a pure domain interface.

IView

IView<S, E>: I_View<S, S, E>
  • Defined in lib/domain/view.ts:48

IView represents the event handling algorithm, responsible for translating the events into denormalized state, which is more adequate for querying.

param evolve

A function/lambda that takes input state of type Si and input event of type Ei as parameters, and returns the output/new state So

param initialState

A starting point / An initial state of type So
