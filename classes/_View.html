Class _View<Si, So, E>

_View is a datatype that represents the event handling algorithm, responsible for translating the events into denormalized state, which is more adequate for querying.

param evolve

A function/lambda that takes input state of type Si and input event of type Ei as parameters, and returns the output/new state So

param initialState

A starting point / An initial state of type So
